# postgretutoreal
applying Postgres16 DB on HTML page

## 목적
1. **postgresql 사용해본다**
2. 바닐라 자바스크립트로 간단한 백엔드 구현해본다

## 개발환경
1. WSL2 Ubuntu 20.04 LTS 에서 작업
2. pg 라는 postgresql DB 파싱 api 사용

## 배운것
1. postgresql 설치 및 세팅
2. postgresql ubuntu cli 환경에서 접속 방법
3. 백엔드 바닐라 JS 복습

## 더 할 것
